# vscode 快捷键整理

| 操作                           | 说明             |
| ------------------------------ | ---------------- |
| Shift + Option + 向上/向下箭头 | 复制一行         |
| Command + X                    | 剪切一行         |
| Command + L                    | 选中整行         |
| Command + B                    | 关闭侧边栏       |
| Cmd + Shift + F                | 所有文件查找     |
| control+d                      | 向后删除字符     |
| cmd+shift+o                    | 跳转到指定方法名 |
| Fn + ← / →                     | 位到行首尾       |
